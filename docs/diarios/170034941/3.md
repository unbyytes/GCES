### Implementação Técnica

**Pipeline final implementada:**
```yaml
stages:
  - security

security_scan:
    stage: security
    image:
        name: python:3.9.19-slim-bookworm
        pull_policy: if-not-present
    before_script:
        - pip install bandit==1.7.5
    script:
        - bandit -r . -ll
    allow_failure: true
    tags:
        - k8s

```
